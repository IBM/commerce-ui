Catalog

  • To create a new catalog object, click the black arrow beside the Create Newicon on the toolbar; then click the type of object.
  • To work with an exesting catalog object, in the explore view to the left, navigate to the object.
  • To search for a catlog object:
  1. 1. In the find area at the top right, type a keyword in the search field. use the asteric (*) as a wildcard.
  2. 2. To select what you want to search for, click the black arrow beside the search icon.
  3. 3. Click the searh icon.
